# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

The Pizza Sales Analysis Dashboard is an interactive business intelligence project developed using Microsoft Power BI. This dashboard transforms raw pizza sales data into meaningful insights, helping businesses understand customer behavior, monitor sales performance, and identify growth opportunities.

By leveraging data visualization and analytics, the dashboard enables stakeholders to make informed decisions based on key performance indicators (KPIs) and sales trends.

---

## 🎯 Project Objectives

- Analyze overall sales performance and revenue generation
- Track total orders and average order value
- Identify top-selling pizzas and categories
- Discover peak sales periods and customer purchasing patterns
- Provide actionable insights through interactive visualizations

---

## 📊 Key Performance Indicators (KPIs)

- 💰 Total Revenue
- 📦 Total Orders
- 🍕 Total Pizzas Sold
- 📈 Average Order Value
- 🏆 Best-Selling Pizza Categories
- ⭐ Top Performing Products
- 📅 Daily and Monthly Sales Trends

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset
- Data Cleaning & Transformation
- Data Visualization

---

## 📂 Dataset Information

The dataset contains transactional pizza sales records with details such as:

- Order ID
- Order Date
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity Sold
- Unit Price
- Total Sales

The data was cleaned, transformed, and modeled before creating the dashboard.
